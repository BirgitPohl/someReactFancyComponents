## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Production

```bash
#It might crash because of some very insecure fetching of data.
npm run build
#or
yarn build
```
```bash
npm run start
#or
yarn start
```
